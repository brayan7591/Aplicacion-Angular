# Aplicacion

Este proyecto fue creado utilizando [Angular CLI](https://github.com/angular/angular-cli) version 21.0.4.

Se trata de un proyecto realizado por diversion y aprendizaje en Angular, donde se encuentra dos versiones de las tablas de verdad y dos versiones de un ajedrez (uno completo y el otro con solo los movimientos) sin utilizar inteligencia artificial.

## Development server

Para ejecutarlo de manera local se necesita el siguiente comando:

```bash
ng serve
```
